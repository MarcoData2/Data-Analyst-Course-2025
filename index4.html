Publica tu Página Web con GitHub Pages

Guía completa para alojar tu sitio de forma gratuita

Crear una cuenta en GitHub

Si no tienes una, ve a github.com y regístrate. Es completamente gratis.

Crear un nuevo repositorio

Haz clic en el botón "+" en la esquina superior derecha y selecciona "New repository".

Nombra tu repositorio: nombreusuario.github.io (reemplaza "nombreusuario" con tu nombre de usuario de GitHub).

Esto es importante para que GitHub Pages funcione correctamente.

Subir los archivos de tu página

Una vez creado el repositorio, puedes arrastrar y soltar tu archivo HTML o usar Git desde la terminal:

# Inicializar repositorio local git init # Agregar archivos git add . # Hacer commit git commit -m "Primera versión de mi página" # Agregar el repositorio remoto git remote add origin https://github.com/tuusuario/tuusuario.github.io.git # Subir los archivos git push -u origin main

Activar GitHub Pages

Ve a la pestaña "Settings" de tu repositorio, luego en el menú lateral izquierdo selecciona "Pages".

En la sección "Source", selecciona la rama "main" (o "master") y haz clic en "Save".

Acceder a tu página

Tu página estará disponible en: https://tuusuario.github.io

Puede tomar unos minutos para que esté disponible después de activar GitHub Pages.

Consejo importante

Asegúrate de que tu archivo principal se llame index.html para que GitHub Pages lo reconozca como la página principal.

¡Y eso es todo! Tu página estará en línea de forma gratuita.

¿Problemas con GitHub Pages?

Si tienes dificultades, puedes:

  • Verificar que el nombre del repositorio sea correcto
  • Esperar unos minutos (a veces tarda en activarse)
  • Consultar la documentación oficial de GitHub Pages
