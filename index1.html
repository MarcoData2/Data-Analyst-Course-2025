¿Qué es SQL?

SQL (Structured Query Language) es el lenguaje estándar para gestionar y manipular bases de datos relacionales. Con SQL puedes crear, modificar y consultar bases de datos de manera eficiente.

¿Por qué aprender SQL?
  • Habilidad muy demandada en el mercado laboral
  • Fundamental para análisis de datos y business intelligence
  • Usado en la mayoría de sistemas de bases de datos
  • Complementa otros lenguajes de programación
  • Útil para cualquier profesional tecnológico

Descargar MySQL

Primeros Pasos con SQL

Instalar MySQL Workbench

Descarga e instala MySQL Workbench desde el sitio oficial de MySQL. Es una herramienta gratuita y muy completa para trabajar con bases de datos MySQL.

Crear tu primera base de datos

Comienza creando una base de datos simple para practicar:

CREATE DATABASE mi_primera_base; USE mi_primera_base;

Crear una tabla

Las tablas almacenan tus datos en filas y columnas:

CREATE TABLE clientes (   id INT PRIMARY KEY AUTO_INCREMENT,   nombre VARCHAR(50),   email VARCHAR(100),   fecha_registro DATE );

Insertar datos

Añade algunos registros a tu tabla:

INSERT INTO clientes (nombre, email, fecha_registro) VALUES   ('Ana García', 'ana@email.com', '2023-05-15'),   ('Carlos López', 'carlos@email.com', '2023-05-20');

Consultar datos

Recupera datos usando la sentencia SELECT:

-- Seleccionar todos los campos SELECT * FROM clientes; -- Seleccionar campos específicos SELECT nombre, email FROM clientes; -- Filtrar resultados SELECT * FROM clientes WHERE fecha_registro > '2023-05-18';
